# 使用指南

简体中文 | [English](../en/user-guide.md) | [文档首页](README.md)

## 功能页面

### 设备

设备页包含设备选择、保持比例的实时画面、控制模式、旋转、鼠标直接触控、硬件按键
和设备检查器。检查器提供 Lockdown 设备信息、已安装应用、IPA 安装与卸载，以及描述
文件列表。

检查器的信息标签页还会通过 Diagnostics Relay 刷新经过隐私过滤的电池诊断，包含电量
与充电状态、循环次数、满充/设计容量、电压/电流、剩余时间和适配器功率。电池及适配器
序列号不会通过私有 API 返回。

画面工具栏始终显示 Home、锁屏、音量加、音量减、静音、Siri 和 Action。标题工具栏
提供窗口置顶、检查器显示和全屏控制。隐藏检查器只会增加画面可用空间，不会拉伸画面。
切换页面和全屏状态会释放所有输入。
画面工具栏会显示当前已保存的控制配置，无需进入映射编辑器即可切换；设备画面全屏的
控制栏中也保留此入口。

### 键盘映射

映射页包含放置画布、映射检查器、硬件快捷键和配置管理。编辑器显示源分辨率和按比例
适配后的显示分辨率。背景可选择实时画面或方向正确的冻结截图；截图可以保存为 PNG，
设备断开后仍可用于离线编辑。

在画布上拖动控件，在检查器中编辑属性，然后保存配置。即使配置包含更多映射，运行时
同时最多选择五个不重复触点身份。

### 应用设置

设置页包含外观、语言、窗口行为、设备音频、剪贴板同步、自动检查更新开关、手动“立即检查”、当前
应用版本和 GitHub 仓库入口。设备音频默认关闭，启用后需重新连接设备生效；播放静音和
音量会立即生效，设备控制工具栏也提供静音按键。剪贴板同步同样默认关闭并在重新连接设备
后生效；启用后，在电脑或设备任一端复制的文本与图片会替换另一端的剪贴板内容。
每次同步会短暂提示方向与内容类型；文本预览会合并空白并限制为 48 个字符。

“设备信息”标签页提供需要二次确认的“重新启动设备”和“关闭设备”操作。两者都会结束
当前控制会话；重启后需等待 iOS 与 USB 服务恢复，关机后则必须手动开机。

### 性能

性能工作台将设备侧 DVT 遥测与桌面视频管线指标放在同一界面。设备指标包含系统 CPU、
进程数、Core Animation FPS 和 GPU 内存；管线指标包含源帧率、解码与显示帧率、传输
带宽、JPEG 编码耗时和帧年龄。
系统 CPU 是将 DVT 报告的聚合 `CPU_TotalLoad` 除以设备逻辑 CPU 核心数后得到的 iPhone
全机百分比，不是桌面主机或单进程 CPU 占用。
“高负载进程”显示 CPU 前十与物理内存前十的并集，可按 CPU 或内存排序。进程 CPU 使用
与系统 CPU 相同的逻辑核心归一化，因此 100% 表示整台设备的总处理能力，而不是单个核心
满载。

只有打开性能工作台时才会启动采样，离开页面后立即停止，避免监控持续增加设备负载。
服务健康区域显示虚拟定位、系统监控和图形监控处于连接、可用、恢复、不可用或停止
状态。单项服务重连不会重建设备视频或输入会话。

## 控制模式

使用设备画面上方的“映射 / 键盘透传”分段控件切换模式。

- **映射**：将物理按键路由到触控映射和硬件按键快捷键。
- **键盘透传**：禁用映射，将 HID 按下和释放事件直接发送到 iOS。支持修饰键、方向键、
  导航键、F1-F24 和数字键盘。输入前需要聚焦设备画面。

`Ctrl+Shift+K` 可快速切换模式。切换模式或页面、窗口失焦、进入全屏或设备断开时，
应用会释放全部触点、硬件按键和键盘 usage，避免输入卡住。

键盘透传发送物理 HID 键，不负责文本组合。CJK 输入法和任意文本输入目前应使用剪贴板
同步，直到项目加入专用文本输入协议。

## 直接触控与方向

鼠标输入只在按比例适配后的可见画面内归一化，不包含黑边或画布空白区域。旋转时宽高
使用相同比例，因此横屏和竖屏触控坐标都会与实际画面对齐。

一份 Universal HID report 最多携带五个类型化触点。Rust 后端会在发送到设备前拒绝
重复身份和越界坐标。

## 配置与 scrcpy-mask 兼容性

配置以经过校验的 JSON 文件保存在 Tauri 应用数据目录。支持选择、激活、新建、复制、
重命名、删除、导入和导出；当前激活配置不能删除。

配置可以关联 App Bundle ID。从 DeviceHub Mask 的 App 列表启动已关联 App 时，会自动
激活对应的已保存控制配置。App 列表会显示当前关联，并可将尚未关联的 App 一键关联到
当前配置，或解除当前配置的关联。跨配置关联和冲突需要在配置编辑器中明确处理。

DeviceHub Mask 可以导入和导出 scrcpy-mask `0.0.1` JSON。全部十三种控制器类型均会
保留，包括嵌套序列坐标、绑定、释放模式、计时和脚本字段。配置兼容不代表支持 Android
传输。

硬件快捷键随配置保存。点击快捷键输入框后按键即可绑定，按 Backspace 或 Delete 清除。
触控映射和硬件按键之间不能共用同一个键，两个硬件按键也不能使用同一个键。Siri 等
功能会保留按住时长。

## 应用与描述文件

应用标签页优先通过 CoreDevice AppService 列出应用，必要时回退到 Lockdown
Installation Proxy。AppService 进程数据可用时会标记运行中的 App，并支持停止或重启。
此外还支持通过原生文件选择器选择 IPA、显示安装阶段，以及卸载用户确认的可移除应用。
App 行接近可视区域时才会通过 SpringBoardServices 按需读取真实图标；图标不可用时继续
显示首字母占位，不影响任何管理操作。

当 App 通过 iOS 文件共享开放 Documents 时，点击应用行的文件夹按钮可进入“App 文档”
工作台。这里可以浏览目录、上传新的本地文件、下载普通文件、新建文件夹、重命名，以及
删除文件或空文件夹。上传不会覆盖同名项目，需要先重命名已有项目或本地文件；House
Arrest 会把所有操作限制在该 App 对外开放的 Documents 根目录内。

卸载前后端会再次查询设备元数据，不会把前端状态作为授权依据。切换设备或结束会话会
取消正在进行的安装或卸载。IPA 上传阶段暂时显示不确定进度，直到 Installation Proxy
开始报告设备端安装百分比。

描述文件通过 Misagent 读取并作为 CMS SignedData 解码。界面只接收标准化元数据，不会
收到原始文件或设备标识符。损坏的描述文件会作为单独错误行显示。

“崩溃报告”标签页会刷新并递归列出 CrashReportCopyMobile 暴露的报告，支持按报告名称和
设备路径搜索。导出使用原生保存对话框，把选中的普通文件写入主机目标路径。浏览保持
只读：应用不会删除设备报告，也不会把报告正文发送到 WebView；路径穿越、非文件条目和
超过 128 MiB 的报告都会被拒绝。

## 设备日志

“设备日志”工作台显示当前 iPhone 的实时 SyslogRelay 日志流，支持实时搜索、暂停显示、
自动滚动、复制当前可见结果以及清空有界内存缓冲区。日志只在该工作台打开时采集，离开
页面后会停止；当日志量过大导致旧条目离开缓冲区时，界面会显示警告。

## 语言与字体

界面支持简体中文（`zh-CN`）和英语（`en-US`）。首次运行时，`zh-*` 系统区域使用中文，
其他区域使用英语。语言切换立即生效，并保存在 `devicehub-mask.locale`。

React 界面和 Ant Design 都使用系统字体。切换语言不会重写已有配置名称或用户标签。

## 更新

可在设置页关闭自动 nightly 检查，手动检查按钮始终可用。确认更新后，应用会下载、验证
签名、安装并重启。签名和发布机制请查看[发布与更新](distribution.md)。

## MCP 自动化

DeviceHub Mask 运行期间，MCP 客户端可以连接 Streamable HTTP 端点
`http://127.0.0.1:8009/mcp`。服务提供截图、点击、滑动、文本与按键输入、硬件按键、
同步多点触控、App 查询、启动/重启与停止、旋转、设备选择与重连、虚拟定位、画面帧同步
和会话状态等工具。

发送坐标输入前应先截图，并把返回的 `image_width` 和 `image_height` 传给 `tap` 或
`swipe`、`multi_touch`，这样截图缩放后坐标仍然准确。对延迟敏感的游戏操作可关闭动作
的 `wait_for_settle`，再把返回的 `frame_version_after` 传给 `wait_for_frame`，等待新画面
后继续截图。MCP 复用桌面应用当前的设备会话，不会再建立一条与手机竞争的连接。

例如，`multi_touch` 可以在同一段 250ms HID 手势中推动左侧摇杆并按住右侧动作按钮：

```json
{
  "contacts": [
    { "x1": 180, "y1": 700, "x2": 240, "y2": 650 },
    { "x1": 850, "y1": 680, "x2": 850, "y2": 680 }
  ],
  "duration_ms": 250,
  "image_width": 1024,
  "image_height": 768
}
```

该端点没有鉴权。除非主机位于可信隔离网络，否则应保持监听回环地址。开发者可用
`DEVICEHUB_MCP_ADDR` 修改监听地址，详见[开发指南](development.md)。
